## Export Checklist

### Pre-Export Validation
- [ ] All screens built with required states
- [ ] All components use design tokens (no hard-coded values)
- [ ] Color contrast meets WCAG 2.1 AA (≥4.5:1 text, ≥3:1 UI)
- [ ] Focus states defined for all interactive elements
- [ ] Auto Layout applied to all frames
- [ ] Naming conventions followed

### Export Process
- [ ] Select all screen frames in Figma
- [ ] Export settings: JPG, 85% quality, 2x scale, sRGB
- [ ] Verify file naming matches convention
- [ ] Organize exports into subdirectories (public, standard-user, admin, modals)
- [ ] Verify all 81 files exported successfully

### Post-Export Validation
- [ ] All files present (81 total)
- [ ] File sizes reasonable (target: 200-500KB per file)
- [ ] Image quality acceptable at 2x scale
- [ ] No rendering artifacts or compression issues
- [ ] Filenames match manifest exactly

- Use this manifest to track export progress
