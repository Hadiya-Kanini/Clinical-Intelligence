## Story ID
   * ID Format: US_091

## Story Title
   * Implement core app navigation and page scaffolding

## Description
  * As a user, I want intuitive navigation with maximum 3 clicks to any feature, so that the app is easy to use and efficient.

## Acceptance Criteria
  * Given the app, When navigating, Then any feature is reachable within 3 clicks (UXR-013).
  * Given Standard User, When authenticated, Then Dashboard (FR-117), Upload (FR-118), Document List (FR-119), and Profile (FR-121) pages are accessible.
  * Given navigation, When displayed, Then breadcrumbs show current location.
  * Given the sidebar, When displayed, Then it shows role-appropriate menu items.

## Edge Cases
   * What happens when navigation state is lost (browser refresh)?
   * How does the system handle deep links?
   * What happens when a user tries to access a page they don't have permission for?
