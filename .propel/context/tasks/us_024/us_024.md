## Story ID
   * ID Format: US_024

## Story Title
   * Implement forgot password page with email input

## Description
  * As a user, I want a forgot password page where I can enter my email to receive a password reset link, so that I can recover my account.

## Acceptance Criteria
  * Given I'm on the login page, When I click "Forgot Password" link, Then I'm navigated to the Forgot Password page (SCR-002).
  * Given I'm on the Forgot Password page, When I enter my email and submit, Then the form validates email format.
  * Given a valid email is submitted, When processed, Then a confirmation message is displayed (FR-115b).
  * Given the forgot password page, When displayed, Then it includes a link back to the login page (FR-115c).

## Edge Cases
   * What happens when the email doesn't exist in the system? (Generic response per FR-009q)
   * What happens when the email service is unavailable?
   * How does the UI handle very long email addresses?
