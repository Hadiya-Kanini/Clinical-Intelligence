## Story ID
   * ID Format: US_039

## Story Title
   * Send credential email to newly created users

## Description
  * As an admin, I want newly created users to receive their login credentials via email, so that they can access the system securely.

## Acceptance Criteria
  * Given a user is successfully created, When creation completes, Then an email with login credentials is sent to the user.
  * Given the credential email, When sent, Then it includes the user's email and a temporary/initial password.
  * Given email sending, When attempted, Then it uses TLS for secure transmission.
  * Given email delivery, When status is known, Then the admin is informed of success or failure.

## Edge Cases
   * What happens when email delivery fails?
   * How does the system handle the case where the email address is invalid?
   * Should the user be required to change their password on first login?
