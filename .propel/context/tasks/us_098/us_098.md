## Story ID
   * ID Format: US_098

## Story Title
   * Export patient data to CSV, JSON, and clipboard

## Description
  * As a clinician, I want to export finalized patient data in CSV, JSON, or clipboard format, so that I can use it in other systems.

## Acceptance Criteria
  * Given finalized data, When export is triggered, Then CSV format option is available (FR-083).
  * Given finalized data, When export is triggered, Then JSON format option is available (FR-084).
  * Given finalized data, When export is triggered, Then clipboard copy option is available (FR-085).
  * Given export completion, When successful, Then PATIENT_DATA_EXPORTED event is logged.

## Edge Cases
   * What happens when export file is very large?
   * How does the system handle clipboard copy on different browsers?
   * What happens when the download fails?
