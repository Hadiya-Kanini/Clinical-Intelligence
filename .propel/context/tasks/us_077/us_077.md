## Story ID
   * ID Format: US_077

## Story Title
   * Display conflict resolution UI with side-by-side comparison

## Description
  * As a clinician, I want to see conflicts side-by-side with their source documents, so that I can make informed resolution decisions.

## Acceptance Criteria
  * Given conflicts exist, When Patient 360 is displayed, Then conflicts are prominently highlighted (UXR-028).
  * Given a conflict, When viewed, Then conflicting values are shown side-by-side with source documents (UXR-029, FR-060).
  * Given conflict display, When rendered, Then each value shows its source citation.
  * Given the conflict UI, When displayed, Then it includes the conflict count badge.

## Edge Cases
   * What happens when there are many conflicts (10+)?
   * How does the system handle conflicts with more than 2 values?
   * What happens when source documents are very long?
