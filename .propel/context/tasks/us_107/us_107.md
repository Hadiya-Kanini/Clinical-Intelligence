## Story ID
   * ID Format: US_107

## Story Title
   * Export audit logs to CSV

## Description
  * As an admin, I want to export filtered audit logs to CSV format, so that I can analyze them externally or share with compliance teams.

## Acceptance Criteria
  * Given filtered logs, When export is triggered, Then CSV file is generated and downloaded (UXR-046).
  * Given export, When processing, Then a progress indicator is shown.
  * Given the CSV, When generated, Then it includes all visible columns and filter criteria.
  * Given large exports, When triggered, Then they are handled efficiently without timeout.

## Edge Cases
   * What happens when export is very large (100k+ rows)?
   * How does the system handle export during high load?
   * What happens when the download is interrupted?
