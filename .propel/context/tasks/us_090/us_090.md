## Story ID
   * ID Format: US_090

## Story Title
   * Display coding outputs section in Patient 360

## Description
  * As a medical coder, I want to see coding outputs in the Patient 360 Dashboard, so that I can review and approve suggested billing codes.

## Acceptance Criteria
  * Given Patient 360, When displayed, Then coding outputs section shows suggested ICD-10 codes (FR-074).
  * Given Patient 360, When displayed, Then coding outputs section shows suggested CPT codes.
  * Given coding section, When displayed, Then Accept/Reject buttons are available for each code.
  * Given code review, When completed, Then status is reflected in the UI.

## Edge Cases
   * What happens when no codes are suggested?
   * How does the section handle many codes (20+)?
   * What happens when code mapping is still in progress?
