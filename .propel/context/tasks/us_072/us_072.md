## Story ID
   * ID Format: US_072

## Story Title
   * Implement side-by-side verification view

## Description
  * As a clinician, I want a side-by-side view of source document and extracted data, so that verification is efficient and intuitive.

## Acceptance Criteria
  * Given Patient 360 view (SCR-008), When displayed, Then source PDF is on left and extracted data on right (UXR-023).
  * Given the split view, When scrolling, Then synchronized scrolling is available.
  * Given data categories, When displayed, Then they have clear visual hierarchy with grouped sections (UXR-027).
  * Given the layout, When on desktop, Then it adapts to viewport width (1280px+).

## Edge Cases
   * What happens with very wide or narrow screens?
   * How does the system handle documents with many pages?
   * What happens when the PDF viewer fails to load?
